
# Professor Frisby's Functional Programming Exercises With Jspm, SystemJs, Mocha and Chai
Amarillo Mike

## Overview
I really enjoyed __Professor Frisby's Mostly Adequate Guide To Functional Programming__ (ebook).

This project contains Professor Frisby's Functional programming examples and exercises configured
with jspm, systemjs, mocha and chai (for testing). The tests are executed in the browser using index.html.
